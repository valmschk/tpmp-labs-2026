# Проект: Запись и чтение текстовых файлов (Task 4)

## Overview
Данный проект предназначен для обработки текстовых данных на языке C. Программа считывает информацию из файла, выполняет обработку в соответствии с алгоритмом и сохраняет результат в новый файл. Проект организован согласно модели КИС.

## Author
Мещенко Валерия, студентка 10 группы.

## Usage
Для сборки проекта используйте команду:
`make`

Для запуска программы:
`./src/prog`

## Additional Notes
- Проект использует автоматическую сборку через GitHub Actions.
- Все текстовые файлы для тестов находятся в корне проекта.
